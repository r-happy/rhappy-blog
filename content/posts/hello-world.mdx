## はじめに
このブログは2023-12-06に作ったブログです。  
とりあえず簡単にブログを作りたいなって思ったので[shadcn/next-contentlayer](https://github.com/shadcn/next-contentlayer)を使って始めました。
選んだ理由はシンプルで綺麗だからというのと自分でいじるのも楽そうだなって思ったからです。

## 始め方
[Githubのリポジトリ](https://github.com/shadcn/next-contentlayer)に行きます。
右上のUse this templateを押して自分のところにもリポジトリを作りクローンもしちゃいましょう。
そしたらそのフォルダに移動し以下のコマンドを実行します。pnpmが入っていない人は[こちら](https://pnpm.io/ja/installation)を参照してください。
```
pnpm install
```
次に以下を実行します。
```
pnpm run dev
```
ブラウザで
[http://localhost:3000](http://localhost:3000)
にアクセスするとブログが起動されいるはずです。

## 記事の追加の仕方
`content`フォルダの中に`pages`と`posts`フォルダがあります。記事は`posts`フォルダにmdxファイルを書くことで追加できます。
